# 세션 인계 문서

- 문서 목적: 새 세션이나 새 환경에서 이전 작업 상태를 빠르게 복원할 수 있도록 현재 기준 상태를 요약한다.
- 범위: 진행 중 작업, 차단 작업, 최근 완료 작업, 잔여 작업, 환경별 검증 현황
- 대상 독자: 개발자, 운영자, 리뷰어, 문서 작성자
- 상태: draft
- 최종 수정일: YYYY-MM-DD
- 관련 문서:

## 1. 현재 작업 요약

- 현재 기준선:
- 예: `로컬 테스트 통과, 문서 허브 갱신 완료`
- 현재 주 작업 축:
- 예: `문서 동기화 자동화 1차 도입`
- 최근 핵심 기준 문서:
- 예: `global_workflow_standard.md`, `PROJECT_PROFILE.md`

## 2. 진행 중 작업

- 현재 `in_progress` 작업:
- 예: `TASK-014 문서 링크 검사 도구화`

## 3. 차단 작업

- 현재 `blocked` 작업:
- 예: `staging 접근 권한 대기`

## 4. 최근 완료 작업

- 최근 완료 작업 목록:
- 예: `TASK-012 handoff 템플릿 도입 완료`
