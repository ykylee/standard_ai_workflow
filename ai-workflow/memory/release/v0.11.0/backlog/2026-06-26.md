## Runtime layer 변경

### TASK-V1110-001: `workflow_kit/common/purpose_ingest.py` 신규 (≈ 180 line)

**Helper functions (5 함수)**:
1. `extract_raw_purpose(purpose_path: Path) -> RawPurposeExtract`:
   - PURPOSE.md 파일 read + frontmatter (`purpose_version`, `last_purpose_review`) parse + §1~§4 본문 raw 추출
   - 1차 정형화 없이 raw text 그대로 (LLM ingest 가능 형태)
   - 부재 시 graceful skip → `RawPurposeExtract(missing=True, ...)` 반환
2. `build_structured_purpose(raw: RawPurposeExtract, workspace_root: Path) -> StructuredPurpose`:
   - Goals G1+ 파싱 (`G1:`, `G2:`, ...) + Key Questions Q1+ 파싱 + Research Scope 포함/제외 sub-section split + Evolving Thesis 본문
   - Pydantic v2 `StructuredPurpose` dataclass: 4 field (goals: list[str], questions: list[str], included_scope: list[str], excluded_scope: list[str], thesis: str, version: int, last_review: date)
   - 1차 정형화 결과 검증 (모든 4-element non-empty) + 위반 시 `StructuredPurposeValidationError` raise (caller 가 catch)
3. `emit_cot_trace(raw: RawPurposeExtract, structured: StructuredPurpose) -> CoTTrace`:
   - CoT (Chain-of-Thought) 2-step trace 기록: `step1_raw_excerpt: str` (≤800 char) + `step2_structured_summary: str` (Goals G1 first + Key Questions count + Scope included count)
   - LLM 의 *directional intent* vs *structural rules* 일관성 verify prompt 생성용
4. `cross_reference_validate(structured: StructuredPurpose, workspace_root: Path) -> CrossRefValidationResult`:
   - `workflow_kit.common.wiki_cascade.file_to_stem` 와 정합 — PURPOSE.md 본문의 mention (e.g. `[[some-concept]]`) 과 실제 `ai-workflow/wiki/concepts/` 파일명 매칭
   - mismatch 발견 시 `CrossRefValidationResult(missing_refs: list[str])` 반환 (advisory, blocking ❌)
5. `run_two_step_cot_ingest(purpose_path: Path, workspace_root: Path) -> TwoStepCoTIngestResult` — unified entry:
   - 4 함수 호출 → raw + structured + cot_trace + cross_ref 합쳐 `TwoStepCoTIngestResult` 반환
   - 모든 stage 의 graceful skip 보장 (raw 부재 시 structured/coT 도 skip, advisory warning)

**Dataclass 4개**: `RawPurposeExtract` / `StructuredPurpose` / `CoTTrace` / `CrossRefValidationResult` / `TwoStepCoTIngestResult`
