## Runtime layer 변경

### TASK-V1111-001: `workflow_kit/common/purpose_graph.py` 신규 (≈ 280 line)

**Helper functions (5 함수)**:
1. `extract_goal_keywords(purpose_path: Path | None) -> list[str]`:
   - PURPOSE.md §1 Goals 본문 의 `G1`, `G2`, ... identifier + 본문 keyword (한국어 명사 / 영어 단어) 추출
   - 부재 시 empty list (graceful skip)
2. `parse_recent_done_items(state_path: Path | None) -> list[RecentDoneItem]`:
   - state.json 의 recent_done_items 30-33 entries 파싱 → `RecentDoneItem(version, commit_hash, summary, deliverable_keywords)` dataclass
   - 부재 / json 파싱 실패 시 graceful skip
3. `compute_goal_coverage(goal_keywords, recent_items) -> GoalCoverageResult`:
   - 각 Goal keyword ↔ recent_done_items summary 의 keyword 매칭률 (Jaccard similarity)
   - covered (≥1 match) / partial (keyword partial match) / uncovered (0 match) 분류
   - 결과: `covered_count / total_count / uncovered_goals: list[str] / coverage_pct: float`
4. `find_surprising_deliverables(goal_keywords, recent_items, scope_excluded) -> SurprisingResult`:
   - recent_done 중 Goals 매핑 0 + scope_excluded 매칭 ❌ 인 deliverable 식별 (scope creep 또는 의도된 out-of-scope 작업)
   - 결과: `surprising: list[str] (deliverable summaries) / is_scope_creep: list[bool] / advisory_warnings: list[str]`
5. `find_gaps(goal_keywords, recent_items) -> GapResult`:
   - Goals 중 recent_done 매핑 0 인 goal 식별 (작업 미진행)
   - 결과: `gaps: list[str] (uncovered goal identifiers) / priority: list[int] (1=highest)`
6. `compute_health_score(coverage, surprising, gaps) -> HealthScore`:
   - 종합 점수: 100 - (uncovered_count × 15) - (scope_creep_count × 10) + min(surprising_count × 5, 25)
   - 0-100 정수, 4 tier (excellent ≥ 80 / good ≥ 60 / fair ≥ 40 / poor < 40)
   - 결과: `score: int / tier: str / breakdown: dict[str, int]`
7. `run_graph_insights(purpose_path, workspace_root, state_path) -> GraphInsightsResult` — unified entry

**Dataclass 4개**: `RecentDoneItem` / `GoalCoverageResult` / `SurprisingResult` / `GapResult` / `HealthScore` / `GraphInsightsResult`
