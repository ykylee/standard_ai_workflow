## Runtime layer 변경

### TASK-V1115-001: decorators.py 2 errors → 0

**Error 분석**:
```
decorators.py:73: Incompatible return value type
  (got "Callable[[F], F]", expected "F")  [return-value]
decorators.py:93: No overload variant of "get" of "dict" matches argument types "int", "str"  [call-overload]
```

**Fix 정공법**:
- line 73: `graceful_shutdown` 의 direct decoration 분기 (`if fn is not None and callable(fn): return _wrap_with_shutdown(fn, ...)`) 가 mypy 가 `Callable[[F], F]` 으로 추론. `cast(F, _wrap_with_shutdown(fn, ...))` 또는 `# type: ignore[return-value]` + 함수 시그니처 `-> Any` 명시.
- line 93: `Signals` Enum 값을 `dict.get(int_key, str_default)` 로 호출 시 overload mismatch. `Signals.SIGINT.value` 로 명시적 int 변환.

### TASK-V1115-002: linter.py 4 errors → 0

**Error 분석**:
```
linter.py:138: "object" has no attribute "__iter__" (set comprehension of backlog.get("in_progress_items", []))
linter.py:139: ... (동일, handoff.get)
linter.py:160: Argument 1 to "len" has incompatible type "object"; expected "Sized"
linter.py:164: ... (동일, done_items)
```

**Fix 정공법**:
- `backlog`, `handoff`, `state` 의 명시적 type annotation: `dict[str, object]` (이미 가능) + `.get(..., [])` 결과의 `isinstance(..., list)` narrow
- 또는 `cast(list[str], backlog.get("in_progress_items", []))`
- `len(done_items)` 는 `done_items = cast("list[object]", handoff.get("recent_done_items", []))` 후 list narrow
